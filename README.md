# HandWriting Recognition

> A Vue.js project

## Build HTML JS

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
## Run python

``` bash
# install dependencies
pip install flask,numpy

# serve with hot reload at localhost:5000
python flaskr

```
